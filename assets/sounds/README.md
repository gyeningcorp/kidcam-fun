# Sound Assets

Place local MP3 sound effects here. All sounds are bundled with the app — no CDN or network fetching.

## Required Files
- `filter_switch.mp3` — Playful "boing" sound when swiping between filters (~0.3s)
- `photo_taken.mp3` — Camera shutter + cheerful chime on capture (~0.5s)
- `stars_sparkle.mp3` — Sparkle/twinkle effect when smile is detected (~1.0s)
- `celebration.mp3` — Short celebration sound on photo taken screen (~1.0s)
- `button_tap.mp3` — Soft tap feedback for UI buttons (~0.1s)
- `parent_gate_success.mp3` — Subtle unlock sound for parent gate (~0.3s)

## Audio Specs
- Format: MP3 (128kbps)
- Duration: Under 2 seconds each
- Volume: Normalized to -14 LUFS
- License: Royalty-free or original
