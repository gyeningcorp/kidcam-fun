# Animation Assets

Place Rive (.riv) animation files here for in-app animations.

## Required Files
- `sparkle_burst.riv` — Star/sparkle burst effect (triggered by smile detection)
- `confetti.riv` — Confetti celebration (shown on photo taken screen)
- `floating_emoji.riv` — Floating emoji background for home screen
- `pulse_ring.riv` — Pulsing ring for the Play button

## Animation Specs
- Format: Rive (.riv)
- Artboard size: 512x512px
- Frame rate: 60fps
- Duration: 1-3 seconds (loopable where appropriate)
- Created with: Rive Editor (rive.app)
