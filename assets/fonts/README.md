# Font Assets

Place Nunito font files here. Nunito is a rounded sans-serif perfect for kid-friendly UI.

## Required Files
- `Nunito-Bold.ttf` — Primary UI text (weight: 700)
- `Nunito-ExtraBold.ttf` — Headings and buttons (weight: 800)
